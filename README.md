# TP-GRUPO-3

Proyecto Integrador

Nuestra principal idea por ahora es una mano que mejora la comunicación de los sordo-mudos

Va a comparar las formas de las manos con el lenguaje de señas y lo va a traducir a un idioma verbal

El objetivo de esto es ayudar a las personas con problemas de comunicación, para mejorar su calidad de vida
La idea es hacer un guante que se coloque en la mano de las personas sordo-mudas y que las ayude a comunicarse sin dificultades.

Vamos a utilizar sensores y un microcontrolador para detectar los movimientos de la mano, y que sean traducidos a un audio (puede ser en ingles o español).

Este guante va a facilitar la comunicación diaria entre personas sordo-mudas y oyentes, se puede usar en escuelas para facilitar la inclusión y tambien para que las personas con estos problemas puedan trabajar como lo puede ser atención al cliente en comercios.

Empezaremos con algunas entrevistas para poder conocer a nuestro publico objetivo

El sistema contará con una interfaz sencilla que permitirá al usuario visualizar en tiempo real la traducción de las señas en texto y audio.

Para mejorar la precisión del reconocimiento, el guante utilizará sensores de flexión y movimiento que registrarán la posición de cada dedo.

Este proyecto busca facilitar la inclusión social de las personas sordomudas, mejorando su comunicación con quienes no conocen el lenguaje de señas.
